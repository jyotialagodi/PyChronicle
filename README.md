# PyChronicle

## Python Execution Visualizer

PyChronicle is a Python-based execution visualizer that traces the execution of a Python program and records variable changes into a SQLite database. It provides multiple visualization tools to understand program execution.

## Features

- Execution Tracer
- Execution Timeline
- Variable History
- Snapshot Viewer
- Execution Summary
- Statistics Viewer
- Export Execution Report
- Clear Database
- Reset Project
- GUI using Tkinter

## Technologies Used

- Python
- Tkinter
- SQLite3
